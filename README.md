# Tic Tac Toe Android App

A simple Tic Tac Toe game developed using **Java** and **XML** in Android Studio.

## Overview
This project is a beginner-level Android application designed to practice core Android development concepts such as UI layout creation, activity handling, and basic game logic implementation using Java.

## Features
- Single Activity architecture
- Clean and minimal user interface
- Two-player gameplay (X vs O)
- Simple and readable Java logic
- Beginner-friendly project structure

## Technology Stack
- Java
- XML
- Android Studio

## Purpose
The purpose of this project is to strengthen foundational Android development skills, including:
- Designing layouts using XML
- Handling user interactions
- Implementing basic game logic in Java

## Future Enhancements
- Add a reset or restart option
- Improve UI and layout design
- Implement single-player mode with basic AI
